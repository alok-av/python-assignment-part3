# Python Assignment - Part 3: File I/O, APIs & Exception Handling

## Theme: Product Explorer & Error-Resilient Logger

This repository contains the solution for Part 3 of the assignment.

### Tasks Completed:
- File Read & Write Basics (`python_notes.txt`)
- API Integration with DummyJSON Products
- Robust API Calls & Exception Handling
- Custom Error Logging (`error_log.txt`)

### Running the Script:
To run the script, first install the required dependencies (if not already installed):
```bash
pip install requests
```

Then, execute the main script:
```bash
python part3_api_files.py
```

> Note: The script is fully interactive. You will be prompted to enter a keyword to search in the notes, and product IDs (1-100) to fetch from the API. Type `quit` when prompted for the product ID to exit the loop and proceed to Task 4.
